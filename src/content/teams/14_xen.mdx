# The Xen Project Hypervisor

The Xen Project Hypervisor Team is responsible for everything related to the Xen Project ecosystem on Nixpkgs. You can find us on:

- The issues tagged `6.topic: xen-project` on [NixOS/nixpkgs](https://github.com/NixOS/nixpkgs/labels/6.topic%3A%20xen-project).
- The [**NixOS with Xen Project Hypervisor** Matrix room](https://matrix.to/#/#xen:nixos.org).

For any issues, comments and enhancements for the Xen packages, please notify the Xen Project Hypervisor Team.

### Responsibilities

- Packaging and testing all `dom0` and `domU` tools.
- Writing documentation for using the Xen Hypervisor on NixOS.
- Implementing the corresponding NixOS module.
- Assuring the security of the Xen Project Hypervisor by rapidly responding to Xen Security Advisories.
