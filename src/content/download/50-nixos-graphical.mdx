You can install NixOS on physical hardware by burning one of the CD images onto a blank CD/DVD disk, or [by copying it onto a USB stick](/manual/nixos/stable#sec-booting-from-usb). For **installation instructions**, please see the [manual](/manual/nixos/stable).

The graphical installation ISO image **contains the graphical NixOS installer** as well as a **Desktop Environment** and several applications. Itâ€™s a _live CD_, so it allows you to get an impression of NixOS (and the Nix package manager) without installing it.

Note that the installer allows installing various desktops, regardless of which desktop image has been booted. The stable image comes in GNOME and Plasma variations, while the unstable image includes both as options in the boot menu.
